DESCRIPTION

r.pack collects raster map elements and support files and compressed them using gzip algorithm for copying. The packed file can be afterwards unpacked by r.unpack.

NOTES

Name of the pack file is determined by default from input parameter. Optionaly the name can be given by output parameter. Currently only 2D raster maps are supported.

EXAMPLE

Pack up raster map aspect into aspect.pack file. 
r.pack input=aspect
the raster map can be afterwards unpacked by 
r.unpack input=aspect.pack

SEE ALSO

r.unpack, r.in.gdal
