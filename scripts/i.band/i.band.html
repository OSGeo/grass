DESCRIPTION

Manages band reference information to a single raster map or a list of raster maps. Band reference can be assigned by bands option. Already assign band reference can be removed by -r flag. This module also allows printing detailed band reference information already assigned to a raster map.

Either a single raster map or a list of raster maps can be given by map option.

EXAMPLES

Assign band reference to single raster map

i.band map=T33UVR_20180506T100031_B01 bands=S2A_1
Note that band references overview can be easily printed by g.bands module.

Assign band reference to list of raster maps

i.band map=T33UVR_20180506T100031_B01,xxx bands=S2A_1

Assign band references to list of raster maps

i.band map=T33UVR_20180506T100031_B01,T33UVR_20180506T100031_B02 bands=S2A_1,S2A_2

Remove band reference

i.band -r map=T33UVR_20180506T100031_B01,T33UVR_20180506T100031_B02

Print band reference information about single raster map

i.band -p map=T33UVR_20180506T100031_B01

Print band reference information about list of raster map

i.band -r map=T33UVR_20180506T100031_B01,T33UVR_20180506T100031_B02
