DESCRIPTION

Prints band references information.

EXAMPLES

Print all band references

g.bands

Filter band references by a shortcut

Only band identifiers related to Sentinel-2A satellite will be printed. 
g.bands bands=S2A

Filter references by a band identifier

Only first band related to Sentinel-2A satellite will be printed. 
g.bands bands=S2A_1

KNOWN ISSUES

g.bands has very limited functionality. Currenlly only system-defined band references can be printed out. The module will be designed in order to manage (add, modify, delete) user-defined band references.
