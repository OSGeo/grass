DESCRIPTION

r.example.segment changes one cell value in a hardcoded location. It is meant to demonstrate how to use the Segment Library together with GRASS raster maps.

EXAMPLE

Create a modified version of the raster map "elevation" (North Carolina sample dataset): 
g.region raster=elevation
r.example.segment input=elevation output=modified_elevation
r.univar elevation
r.univar modified_elevation

SEE ALSO

r.example, r.example.segmulti, v.example Segment Library in the GRASS Programmer's Manual
