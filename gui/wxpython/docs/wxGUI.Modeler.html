DESCRIPTION

Note: wxGUI Modeler is currently under development.

Graphical modeler is a wxGUI extension which allows the user to create, edit, and manage models. The modeler can be launched from the menu "File | Graphical modeler" or by clicking on icon in the toolbar.

The modeler currently allows to:

  • define actions (GRASS modules)
  • define data items (raster, vector, 3D raster maps)
  • define relations between data and action items
  • define loops and conditions (if-else statements)
  • define model variables
  • define intermediate data
  • parameterize actions
  • validate model
  • run model
  • store model settings to the file (GRASS Model File|*.gxm)
  • export model settings to Python script
  • export model to image file
