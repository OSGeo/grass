DESCRIPTION

r.surf.gauss produces a raster map of Gaussian deviates whose mean and standard deviation can be expressed by the user. It uses a Gaussian random number generator. It is essentialy the same as r.surf.random, but uses a Gaussian random number generator instead.

Histogram of map generated with r.surf.gauss (mean=0, sigma=10)

EXAMPLE

r.surf.gauss out=gauss mean=0 sigma=10
