r.li.richness calculates the "richness's diversity index" as:

with:
  • m: number of different patch type

EXAMPLES

To calculate richness's diversity index on map my_map, using my_conf configuration file and saving results in my_out file run:
r.li.richness input=my_map conf=my_conf output=my_out

NOTES

Do not use absolute path names for output files. They are raster if moving window disposition was selected, otherwise they are in ~/.r.li/output folder.
If input raster is full of null values r.li.richness returns -1.
If you want to have null values instead run
