DESCRIPTION

This module exports all maps registered in a space time raster datasets as VTK legacy files using a numerical numbering scheme. The VTK files can be visualized with any VTK based visualize. Our preferred tool is ParaView. The VTK legacy files are created using r.out.vtk.

EXAMPLE

In this simple example we create several raster maps with random values and one elevation map. The random value raster maps are registered in a space time raster dataset. All maps of this space time raster dataset are exported using t.rast.out.vtk into the directory /tmp/export. 
mkdir /tmp/export
t.rast.out.vtk input=precip_abs directory=/tmp/export/ elevation=elevation
