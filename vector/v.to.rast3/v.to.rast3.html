DESCRIPTION

v.to.rast3 converts a GRASS 3D vector point map to a GRASS raster3d map.

NOTES

When converting from a 3D vector point layer to a raster3d map a vector point is converted into a single 3d cell representing the location of the vector point. As 3d cell value the database attribute of the vector point is stored as float.

This screenshot shows the result of the v.to.rast3 test. Visualized are the cube of the grass region, the vector points as black dots and the voxel cells as wireframe model. Only cells with non-null values are shown.
