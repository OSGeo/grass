DESCRIPTION

d.rast displays the specified raster map in the active display frame on the graphics monitor.

EXAMPLE

Display raster map "elevation" (only values between 75 and 80): 
d.rast map=elevation values=75-80

SEE ALSO

d.rast.arrow, d.rast.leg, d.rast.num, d.mon, d.erase, d.vect
