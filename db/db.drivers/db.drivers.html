DESCRIPTION

db.drivers lists all database drivers (DBMI backends).

EXAMPLE

Show all installed GRASS database drivers: 

db.drivers -p
pg
ogr
sqlite
dbf

SEE ALSO

db.connect, db.describe, db.droptable, db.execute, db.login, db.tables, GRASS SQL interface
