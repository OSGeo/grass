DESCRIPTION

nviz_cmd allows users to realistically render multiple surfaces (raster data) in a 3D space, optionally using thematic coloring, draping 2D vector data over the surfaces, displaying 3D vector data in the space, and visualization of volume data (3D raster data) from the command line.

EXAMPLE

Render elevation map in a 3D space. 
g.region rast=elevation
nviz_cmd elevation_map=elevation output=elev perspective=15
