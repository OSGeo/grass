DESCRIPTION

ximgview is a simple X11 image viewer for 32-bpp BMP images, as can be created with the PNG and cairo drivers. The display is continually refreshed.

NOTES

The display driver must be configure to map the file, with GRASS_PNG_MAPPED=TRUE. This ensures that the file will remain a constant size, rather than being truncated whenever it is updated.
