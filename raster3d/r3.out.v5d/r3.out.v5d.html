DESCRIPTION

Exports RASTER3D grids to v5d format. map is a valid RASTER3D grid in the current mapset. output is the name of a v5d file which will be written in the current working directory. VIS5D is a system for interactive visualization of large 5D gridded data sets such as those produced by numerical weather models. One can make isosurfaces, contour line slices, colored slices, volume renderings, etc of data in a 3D grid, then rotate and animate the images in real time. There's also a feature for wind trajectory tracing, a way to make text anotations for publications, support for interactive data analysis, etc.
