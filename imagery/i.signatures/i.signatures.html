DESCRIPTION

i.signatures module allows to manage signature files generated by i.gensig, i.gensigset and i.svm.train. The module can perform multiple actions per run. The order of execution is "copy", "remove", "rename". If no actions are specified, then the module lists (prints) all signature files of specified type.

EXAMPLES

Print names of all signature files: 
i.signatures
Print only signature files of certain type as a JSON: 
i.signatures type=sigset format=json
Delete signature file called "foo" of type "sig" (i.gensig / i.maxlik). 
i.signatures remove=foo type=sig
Copy signature file "bar" from mapset "baz" to current mapset 
i.signatures copy=bar@baz,best_version type=sigset
